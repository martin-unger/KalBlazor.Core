# KalBlazor
KalBlazor is a Blazor component library that provides reusable layout building blocks using TailwindCSS, helping .NET applications to create structured, responsive user interfaces with a consistent component model.

The name ‘Kal’ comes from the Macedonian language and means ‘mud’. It’s a small tribute to MudBlazor, which is an absolutely brilliant Blazor library, but wasn’t quite perfect for our flexible requirements.

That’s why we’ve launched KalBlazor.

## 🚀  Getting Started

First, install Bun:

```powershell
powershell -c "irm bun.sh/install.ps1 | iex"
```

Then build and run the UI project:

```powershell
dotnet run --project UI/UI.csproj
```

There is also an example client you can run:

```powershell
dotnet run --project .\Example\Example.UI\Example.UI\Example.UI.csproj
```
